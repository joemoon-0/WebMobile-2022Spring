ICP 2 - HTML & CSS

This webpage is for ICP 2 of the Web and Mobile Programming class.

Here are the requirements on what this webpage should contain:

  1. paragraphs
  2. headings
  3. unordered list
  4. ordered list
  5. at least two div & span elements
  6. at least four block and inline elements
  7. images at appropriate locations in the HTML file and should be clickable, i.e., when you click the image, it should redirect you to the location of the image in a new tab
  8. any relevant video/YouTube video. The video should have controls like play, pause, etc.
